### Проект Trendy

### Цель проекта: 

Платформа для мониторинга и анализа медиаконтента из социальных сетей как youtube.com, tiktok.com.

---
## Архитектура
Тип репозитория: `монорепозиторий`

Языки программирования: `Go`,`Python`.

### Почему монорепозиторий?
Было принято решение использовать монорепозиторий для всех сервисов,
так как проект небольшой (не очень много сервисов) удобнее всего будет использовать `монорепозиторий`.

### Почему был выбран Go?
Go был выбран для сервисов обработки данных, аутентификации и других системных сервисов. 
Из-за своей быстроты и эффективности.

### Почему был выбран Python?
Python был выбран потому что язык имеет много готовых библиотек которые находятся на постоянной поддержке авторов и если api сервисов измениться, то автор быстро изменит библиотеку. 

На Go нужных библиотек или нет, или их давно не поддерживают или они не актуальны или могут выйти из строя. Плюс на Python есть много примеров использования данных библиотек.


### Как запустить:

Запустить контейнеры:
```bash
docker compose up -d --build 
```
